# Coming Soon

A Next.js project for a coming soon landing page.

## Getting Started

### Prerequisites
- Node.js 16+ 
- npm or yarn

### Installation

```bash
npm install
# or
yarn install
```

### Features

- Lightning-fast performance with Next.js
- Responsive design for all devices
- SEO optimized
- Email subscription ready
- Customizable landing page template

### Running the Project

```bash
npm run dev
```

View the application at [http://localhost:3000](http://localhost:3000).


```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

### Build

```bash
npm run build
npm start
# or
yarn build
yarn start
```
